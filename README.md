# Kinetic Hub MVP

Sitio web base para Kinetic Hub inspirado en la lógica funcional de plataformas de eventos deportivos,
con una dirección visual minimalista, moderna y escalable.

## Páginas incluidas (MVP)

- Home: `index.html`
- Explorar eventos: `eventos.html`
- Detalle de evento: `evento.html`
- Series y categorías: `series.html`
- Blog y noticias: `blog.html`
- Nosotros: `nosotros.html`
- Contacto: `contacto.html`

## Estructura de archivos

- `styles.css`: sistema visual global y componentes reutilizables
- `script.js`: menú responsive, animaciones reveal y filtros de catálogo

## Cómo verlo

1. Abre `index.html` en el navegador.
2. Navega entre secciones y páginas desde el menú superior o footer.

## Alcance

Este repositorio implementa Fase 1 (sitio base / MVP).

Las funciones de plataforma avanzada como login, perfil, resultados, score, historial,
ranking, retos y membresías se presentan como evolución futura, no como funcionalidades activas del MVP.
